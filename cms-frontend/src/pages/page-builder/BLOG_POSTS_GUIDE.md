# Blog Posts Per Section - Usage Guide

## Overview

You can now add blog posts to any section (Hero, Features, Testimonials, Carousel, CTA) using the "Manage Blog Posts" feature, just like you manage content blocks.

## How to Use Blog Posts Per Section

### Step 1: Edit a Section

1. Click on any section you want to add blogs to (Hero, Features, Testimonials, Carousel, or CTA)
2. The section editor panel will open

### Step 2: Access Blog Manager

1. Scroll down to the **"Manage Blog Posts"** section
2. Click the **"+ Add Blog"** button to create a new blog post

### Step 3: Blog Editor Opens

The blog editor will expand with the following options:

#### Blog Information:

- **Blog Title** - Main title of the blog post
- **Subtitle** - Brief description
- **Author** - Name of the author
- **Date** - Publication date
- **Featured Image URL** - Main image for the blog post

#### Styling:

- **Header Background** - Background color with color picker
- **Title Color** - Heading text color
- **Text Color** - Body text color
- **Accent Color** - Used for quotes and dividers

When you create/save a page, blog posts appear:

- **After Carousel** - If you added blogs to Carousel section
- **After CTA** - If you added blogs to CTA section

## Tips

1. **Best Practices:**

   - Use headings to structure your blog
   - Add images between text sections
   - Use quotes for testimonials/key insights
   - Use dividers to separate concepts

2. **Images:**

   - Use high-quality images (800x600px or larger)
   - Always add alt text for accessibility
   - Use image grids for showing multiple related images

3. **Content:**
   - Start with a strong heading
   - Keep paragraphs readable (2-3 sentences each)
   - Use subheadings to break up long content
   - End with a call-to-action

## Troubleshooting

**Blog editor not opening?**

- Make sure you've selected a section (Hero, Features, etc.)
- Click "Manage Blog Posts" button
- Click "+ Add Blog" to create a new blog

**Images not showing?**

- Check that the image URL is correct and accessible
- Use HTTPS URLs when possible
- Ensure the image format is supported (JPEG, PNG, WebP)

**Grid showing wrong columns?**

- 1 image = 1 column (full width)
- 2 images = 2 columns
- 3+ images = 3 columns
- This is automatic and responsive
