# How to Add Text & Image Blocks After Any Section

This guide shows you how to add flexible text and image content after any section in your pages using both the **Classic Page Builder** and **Professional Editor**.

## Method 1: Classic Page Builder (Easiest)

### How to Use:

1. **Build Your Base Section**

   - Create your main section (Hero, Features, Testimonials, Carousel, etc.)

   - **Heading** - Main title
   - **Subheading** - Secondary title
   - **Description** - Main text content
   - **Button** - Optional call-to-action button with URL
   - **Image** - Image URL and alt text

5. **Style It**

   - Background color
   - Text color
   - Font size
   - Padding and spacing
   - Border radius

6. **Add Multiple Blocks**
   - Keep adding blocks as needed - they'll stack below each section
   - Edit or delete them anytime

## Method 2: Professional Editor (Most Flexible)

### How to Use:

1. **Go to Professional Editor**

   - Click "New Page" â†’ Select "Professional Editor" or "Hybrid Mode"

2. **Available Blocks for Text & Image**

   - **Text Block** - Pure text content
   - **Image Block** - Pure images
   - **Heading Block** - Titles and subtitles
   - **Paragraph Block** - Body text
   - **Button Block** - Call-to-action buttons
   - **Grid Block** - Create complex layouts

3. **Create a Text & Image Combination**

   - Add a **Grid Block** with 2 columns
   - Add **Text Block** in one column
   - Add **Image Block** in the other column
   - Style each independently

4. **Or Use Multiple Blocks**
   - Heading block for title
   - Paragraph block for description
   - Button block for CTA
   - Image block below
   - All styled individually

## Best Practices

### 1. **Image Optimization**

- Use web-optimized images (JPG for photos, PNG for graphics)
- Recommended sizes:
  - Side-by-side: 600Ã—400px
  - Full-width: 1200Ã—400px
- Always add descriptive alt text for accessibility

### 2. **Content Tips**

- Keep headings short and punchy (5-10 words)
- Subheading should expand on the heading
- Description: 50-100 words for best readability
- Use button text that's action-oriented ("Learn More", "Get Started", etc.)

### 3. **Styling Tips**

- Use consistent colors across blocks
- Maintain 20-40px padding for breathing room
- Use 4-8px border radius for modern look
- Keep text color contrast high for accessibility

### 4. **Layout Tips**

- Don't mix too many layouts - stick to 2-3 patterns per page
- Use text-left and text-right alternately for visual rhythm
- Use text-top/bottom for mobile optimization
- Use text-full for dramatic statements

### 5. **Adding Buttons**

- Optional but recommended for conversion
- Should link to:
  - Internal pages (/about, /features)
  - External sites (https://example.com)
  - Email (mailto:email@example.com)
- Common CTAs: "Learn More", "Get Started", "Download", "Contact Us"

## Troubleshooting

### Image not showing?

- Check that URL starts with `http://` or `https://`
- Verify the image file exists
- Try a different image URL

### Text overlapping image?

- Choose a different layout (Text Top, Text Bottom, or Text Full)
- Increase padding
- Reduce font size

### Colors look wrong?

- Check background and text color contrast
- Use lighter backgrounds with dark text
- Use darker backgrounds with light text

### Block not appearing?

- Scroll down to see it
- Make sure you're in the right editor mode
- Try refreshing the page
